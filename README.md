1.  **[span_3](start_span)El Sujeto Único:** Solo existe una conciencia real[span_3](end_span). [span_4](start_span)Todo lo demás (objetos, personas, estrellas) son proyecciones generadas (NPCs) para sostener la narrativa de la experiencia[span_4](end_span).
